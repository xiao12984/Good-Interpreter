Good-Interpreter 一键安装包打包说明

本目录提供一键打包脚本和 Inno Setup 安装脚本，用于生成 Windows 安装程序 Good-Interpreter-Setup.exe。

一、打包机准备工具

  • .NET 8 SDK：发布 WinForms 启动器。
  • Python 3 和 PyInstaller：把后端打包为 GoodInterpreter.Backend.exe。
  • Node.js 和 npm：构建 frontend/dist。
  • Inno Setup 6：编译安装包。

二、一键生成安装包

cd D:\code\录写\Good-Interpreter
powershell -ExecutionPolicy Bypass -File .\installer\package.ps1

生成结果：

D:\code\录写\Good-Interpreter\dist\Good-Interpreter-Setup.exe

三、安装包内容

  • 启动器 GoodInterpreter.Launcher.exe。
  • 内置后端 GoodInterpreter.Backend.exe，目标电脑无需安装 Python。
  • 前端静态文件 frontend/dist，目标电脑无需安装 Node.js 或 npm。
  • 火山 AST protobuf 资源 backend/ast_python。

四、目标电脑使用流程

  1. 运行 Good-Interpreter-Setup.exe。
  2. 打开 Good-Interpreter。
  3. 输入火山 App Key 和 Access Key。
  4. 点击“保存并开始使用”。
  5. 程序自动启动服务并打开翻译页面。

安装包默认安装到 %LOCALAPPDATA%\Programs\Good-Interpreter，这样 .env、数据库和日志都能正常写入，不需要管理员权限。
